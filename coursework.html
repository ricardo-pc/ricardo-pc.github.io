UC Berkeley M.A. in Statistics

Coursework highlights the tools and theory I use for data science, with an emphasis on modeling, decision-making, and scalable computation.

In Progress (Current Term)

STAT 214

Data Analysis & ML for Decision Making

Applied machine learning for real-world decision-making. Focuses on model evaluation under uncertainty, causal inference methods, and practical deployment of ML systems for high-impact decisions.

Python Applied ML Causal Inference

STAT 230A

Linear Models

Comprehensive regression theory from OLS and Gauss-Markov theorem to GLMs and generalized estimating equations. Covers heteroskedasticity, cluster-robust inference, model selection, shrinkage methods (Ridge/LASSO), and extensions to categorical and count outcomes.

Python Regression GLMs

STAT 254

Modern Statistical Prediction & ML

Probabilistic machine learning theory covering empirical risk minimization, complexity bounds, and uniform laws. Emphasizes model evaluation via cross-validation, regularization schemes, classification with proxy losses, and connections between linear models and sophisticated ML methods.

Python Deep Learning Optimization

COMPSCI 289A

Introduction to Machine Learning

Theoretical foundations and applications of ML from problem formulation to model deployment. Covers unsupervised learning (k-means, PCA), regression and classification, deep learning architectures (CNNs, RNNs, transformers), and generative models (LLMs, GANs). Includes PyTorch implementation and backpropagation.

Python ML Theory Algorithms

STAT 238

Bayesian Statistics

Bayesian inference framework including prior elicitation, conjugate analysis, and hierarchical modeling. Focuses on MCMC methods (Gibbs sampling, Metropolis-Hastings), modern computational techniques (HMC, variational inference), and applications to complex statistical models.

Python MCMC Hierarchical Models
